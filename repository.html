- Part 2 -
- creating a respitory && opening a Github Account -

Congradulations!, you are now onto the second step. Once you have brainstormed hard enough and have formulated an idea now is the time you implement that idea. First go to Github and create an account. Once done, you click on Make a new Repository where you have the chance to upload your files or code directly on the website. You have the option to make your repository public or private but for now, it doesn't matter. Name the repository whatever you want and now you are ready for step 3!
