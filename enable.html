- enable Github Pages -

Enabling Github Pages for a repository involves a straightforward process. Firstly , access the repository you made, and navigate to the Settings tab. Once there, locate the Pages section in the settings. Choose from the desired branch for GitHub Pages from the source dropdown menu, usually the main or master branch depending on what the external editor you used. Specify the folder from which GitHub Pages will be served, opting for either the root directory or a different folder. Finally save your settings, and GitHub pages will be activated for the repository, a free static website will be made, providing you with a URL to access and share your project through a hosted website.
